Set up Advice:

Problems installing nano on Windows 10 from previous workshops:

A recurrent problem is that some students cannot install nano in Windows 10. If nothing works suggest any other text editor like atom or notepad++ (both opensource). If installed as administrator (with right click - open as admin) everything should work from a new terminal - However, if installed and after opening a new terminal they don't open then the easier solution I've found is to create an alias:

alias notepadpp="C:\Program Files (x86)\Notepad++\Notepad++.exe -multiInst -notabbar -nosession -noPlugin"

for that to persist over different terminal session you should save that into their bashrc:
echo 'alias notepadpp="C:\Program Files (x86)\Notepad++\Notepad++.exe -multiInst -notabbar -nosession -noPlugin"' >> ~/.bashrc
